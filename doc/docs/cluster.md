## 显卡性能概览
深度学习对高性能GPU有非常大的需求, 而AI集群中又有各种版本的GPU, 故在此给出简单测试结果, 供大家参考.

**测试环境**: CUDA9.0 + Pytorch1.2 + VGG16 跑Cifar10  
**分数标准**: 以K80性能为一个基本单位, 可以简单地理解为运算速度, 如GTX 1080分数为11.8, 说明同样的网络K80跑一个epoch, GTX 1080可以跑11.8个epoch.
